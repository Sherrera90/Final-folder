Challenge

Many weather apps look generic, with minimal design differentiation.I needed to balance functionality (clear data, responsive layout) with a retro 90s aesthetic (bold colors, pixel fonts, playful graphics), all while ensuring the design feels nostalgic without sacrificing usability.

Goals

To combine technical skills in API integration and front-end development with a creative visual design. The objective was to deliver real-time weather data in a fun, engaging way that stands out from typical weather apps.

The Approach

1. API Integration

  • Pulled live data for temperation, conditions, "feels like", wind speed, and humidity.
  • Added an hourly forecast with weather icons.

2. UI & Retro Design

  • Gradient style heading with a search bar and pixel-inspired buttons.
  • Use of popular squiggle design often seen on disposable cups in the 90's/
  • Pixel inspired typography and bold neon highlight text.
  • Playful weather icons with a cartoon vibe.

3. User Experience

  • Added the ability to easily search for your city.
  • Large centered temperature and condition that is easily seen.
  • Hourly forecase displayed in a timeline-style with icons and temperatures below it.

Summary

The final design offers live, accurate weather updates in a UI that feels nostalgic and fun. By leaning into 90's web nostalgia, the app stands apart from stand weather apps, offering a functional tool that is wrapped in a visually exciting experience. While showcasing my experience in API calls, responsive CSS and DOM manipulation.
